SimpleOSD is tool writte in C++ for displaying text on top of the desktop.

Usage: simpleOSD [options] text

    Options:
	-v   vertical alignment: 0 = top, 1 = middle, 2 = bottom
	-h   horizontal alignment: 0 = left, 1 = center, 2 = right
	-l   vertical offset
	-b   horizontal offset
	-t   timeout: 0 = forever
	-f   font string
	-c   color name
